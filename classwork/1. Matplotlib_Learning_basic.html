Matplotlib Basics¶

Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python. It has very similar plotting capabilties like MATLAB's.

  • Offical page: https://matplotlib.org/
  • Quick start guide: https://matplotlib.org/stable/users/explain/quick_start.html#quick-start
  • Tutorial: https://matplotlib.org/stable/tutorials/pyplot.html#sphx-glr-tutorials-pyplot-py
  • Sample figures creation: https://matplotlib.org/stable/plot_types/index.html

Anatomy of a matlab figure¶

Coding styles:¶

You can use matplotlib in 2 ways:

  • Implicit interface: Rely on pyplot to implicitly create and manage the Figures and Axes, and use pyplot functions for plotting.

  • Explicit interface: Explicitly create Figures and Axes, and call methods on them (the "object-oriented (OO) style").

Object Oriented method¶

  • More flexible and control
  • Instantitiate fig object
  • Invoke differnt methods as needed

Figure: figure keeps track of all the child Axes
Axes: An axes is an Artist attached to a Figure that contains a region for plotting data
Axis: objects that set the scale and limits and generate ticks (the marks on the Axis) and ticklabels (strings labeling the ticks)
Artist: everything visible on the Figure is an Artist

In [78]:
#Create an empty canvas 
fig = plt.figure()

axes = fig.add_axes([0.1,0.1,0.7,0.7]) #left, bottom, width, height 
axes.plot(x,y,'b+-')
axes.set_xlabel('x label')
axes.set_ylabel('y label')
axes.set_title('OOP example figure')

Fig size and DPI¶

  • dpi:The resolution of the figure in dots-per-inch.

Line and marker styles¶

We can change a number of styling parameters in graph plotting

In [25]:
fig = plt.figure(figsize=(10,5))
ax = fig.add_axes([0,0,1,1])

#Changing different linewidth 
ax.plot(x, x+1, color="red", linewidth=0.25, label='x+1') 
ax.plot(x, x+2, color="red", linewidth=0.50, label='x+2')
ax.plot(x, x+3, color="red", linewidth=1.00, label='x+3')
ax.plot(x, x+4, color="red", linewidth=2.00, label='x+4')

# Changing different line style:  options are: '-', '--', '-.', ':'
ax.plot(x, x+5, color="green", lw=3, linestyle='-',label='x+5')
ax.plot(x, x+6, color="green", lw=3, ls='-.',label='x+6')
ax.plot(x, x+7, color="green", lw=3, ls=':',label='x+7')

# Changing different markers. options are = '+', 'o', '*', 's (squre)', 'd (diamond)', '.', '1', '2', '3', '4', ...
ax.plot(x, x+ 9, color="blue", lw=3, ls='-', marker='+')
ax.plot(x, x+10, color="blue", lw=3, ls='--', marker='o')
ax.plot(x, x+11, color="blue", lw=3, ls='-', marker='s')
ax.plot(x, x+12, color="blue", lw=3, ls='--', marker='d')

# Changing marker size and style
ax.plot(x, x+13, color="purple", lw=1, ls='-', marker='o', markersize=2)
ax.plot(x, x+14, color="purple", lw=1, ls='-', marker='o', markersize=4)
ax.plot(x, x+15, color="purple", lw=1, ls='-', marker='o', markersize=8, markerfacecolor="red")
ax.plot(x, x+16, color="purple", lw=1, ls='-', marker='s', markersize=8, 
        markerfacecolor="yellow", markeredgewidth=3, markeredgecolor="orange");
ax.legend()
plt.show()

Customizing Plot range¶

  • axis('tight): tightly fitted axis
  • custom axis: set_xlim() and set_ylim()
In [20]:
fig, ax = plt.subplots(1,3, figsize=(12,4))

#Deafult plot range 
ax[0].plot(x,x,x,x**2,x,x**3)
ax[0].set_title('Default axis ranges')

#tight plot range 
ax[1].plot(x,x,x,x**2,x,x**3)
ax[1].axis('tight')
ax[1].set_title('Tight axis ranges')

#Custom plot range
ax[2].plot(x,x,x,x**2,x,x**3)
ax[2].set_xlim([0,5])
ax[2].set_ylim([0,60])
ax[2].set_title('Custom axis ranges')

Text(0.5, 1.0, 'Custom axis ranges')

Box-plot¶

Make a box and whisker plot for each column of x or each vector in sequence x. The box extends from the lower to upper quartile values of the data, with a line at the median.

In [32]:
plt.figure(figsize=(6,4))
# generate a nested list with 4 different std values
data = [np.random.normal(0,std,100) for std in range(1,5)] 
plt.style.use('seaborn')

plt.boxplot(x=data)

plt.show()

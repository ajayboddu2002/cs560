Regression plot (lmplot())¶

  1. Plot data and regression model fits across a FacetGrid.
  2. lmplot allows you to display linear models, but it also conveniently allows you to split up those plots based on features

Ref: https://seaborn.pydata.org/generated/seaborn.lmplot.html#seaborn.lmplot

Visualizing Statiscial Relationships¶

Statistical analysis is a process of understanding how variables in a dataset relate to each other and how those relationships depend on other variables.

We will start with the most popular and easy one: scatterplot()

Ref: https://seaborn.pydata.org/tutorial/relational.html
Palette: https://seaborn.pydata.org/tutorial/color_palettes.html

relplot()¶

is a generic relationship plot, in where you can pass scatterplot() and lineplot() through kind attribute

In [8]:
#default kind is scatterplot()
sns.relplot(x='total_bill',y='tip',data=tips)
#sns.despine(top=False,right=False)
Out[8]:
<seaborn.axisgrid.FacetGrid at 0x7f9d9240b880>

Distinguished marker style¶

sns.relplot(x='total_bill',y='tip',data=tips,hue='smoker',style='smoker')

  • Adding one more semantic variable: size which will chage the size of point
  • sizes will normalize from (1 to 6) to (15, 100): to just make the point bigger
  • here size is the party size
In [22]:
